Problem 7: 质因数分解

Problem 7: 质因数分解

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 8  Solved: 1
[Submit][Status][Web Board]

Description

已知正整数n（3＜n＜2,000,000,000）是两个不同的质数的乘积，试求出较大的那个质数。

Input

第一个数为m(0＜m＜=50)，表示后面跟有m个正整数，每个整数间使用空格分隔。

Output

对于每个正整数，输出较大的那个质数。输出的质数间使用空格分隔。

Sample Input

2 4 21

Sample Output

2 7
