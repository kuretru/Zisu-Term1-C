Problem 3: Footnotes

Problem 3: Footnotes

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 1  Solved: 1
[Submit][Status][Web Board]

Description

A footnote appears at the bottom of the page and must be on the same page as the main text they explain. A word in the main text can be marked with a footnote reference mark. There may be one or more lines in a footnote and one or more footnotes in a page. Suppose that the maximum allowable number of lines is constant for pages in a book . That is the lines of the texts within a page can be only equal or less than the constant number. Write a program to find out the minimum number of pages for a given book. Write a program to compute the minimum number of pages for the book.

Input

The first line includes two integers: the number of all lines of a book(N, 2≤N≤1000) and the maximum number of lines per page in the book(K, 2≤K≤1000). The two numbers N and K are separated by spaces. The second line includes one integer: the number of all footnotes in the book (F, 1≤F≤100). The following F lines are formatted as follows: Each line includes two integers X and Y, representing that the footnote appears at the Xth line and contains Y lines. All the footnotes are sorted with the early or late order(按先后顺序).

Output

An integer: the minimum number of pages for the book.

Sample Input

7 3 2 2 1 4 2

Sample Output

4
