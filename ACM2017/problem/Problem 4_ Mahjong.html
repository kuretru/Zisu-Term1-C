Problem 4: Mahjong

Problem 4: Mahjong

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 1  Solved: 1
[Submit][Status][Web Board]

Description

We consider a special mahjong. In this special mahjong, there are only ordinal number tiles(序数牌) ranging from 1 to 9, and each number has no more than 4 tiles. “To win”(胡牌) means a player’s tiles conform to the pattern AA + x * (ABC) + y * (AAA), in which AA/AAA represents two/three identical tiles, and ABC represents three consecutive serial tiles. x and y can be any natural number. “To wait”(听牌) means waiting for the one necessary tile to win. For example, 1 1 1 2 3 3 3，is waiting for 1, 2, 3 or 4. Waiting for 1：111 + 123 + 33; Waiting for 2：111 + 22 + 333; Waiting for 3：11 + 123 + 333; Waiting for 4：111 + 234 + 33. Given a set of tiles, please write a program to determine whether it is to wait or not. The number of tiles in the set is 13-3*k (0≤k≤4).

Input

The first line includes one integer: k The following k lines are formatted as follows: the first integer n represents there are n tiles in the current set and is followed by tiles’ numbers.

Output

If not “to wait”, print “NO!”; otherwise, print all the tiles to wait in an increasing order.

Sample Input

4 4 1 1 3 4 7 1 2 4 5 7 7 8 7 1 1 1 2 3 3 3 10 1 2 3 4 4 5 6 7 8 9

Sample Output

2 5 NO! 1 2 3 4 1 4 7
