Problem 2: 整数分解(N19)

Problem 2: 整数分解(N19)

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 241  Solved: 109
[Submit][Status][Web Board]

Description

将整数n分成k份，且每份不能为空，任意两种分法不能相同（不考虑顺序）。
如n=7,k=3时，下面的几种分法认为是相同的：
1，2，4； 1，4，2； 4，1，2； 2，1，4； 2，4，1； 4，2，1；
请输出所有的分法？

Input

第一个数为m(0＜m＜=20)，表示后面跟有m组测试数据。每组测试数据包括n和k(6＜n＜=100,2＜=k＜=4)。

Output

每组测试数据输出一个所有的分法，每种分法占一行，并按从小到大的顺序输出，每个数据用一个空格分隔。每组测试数据的结果使用一个空行分隔。

Sample Input

2 7 3 7 2

Sample Output

1 1 5 1 2 4 1 3 3 2 2 3 1 6 2 5 3 4
