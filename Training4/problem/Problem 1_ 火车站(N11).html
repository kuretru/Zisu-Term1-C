Problem 1: 火车站(N11)

Problem 1: 火车站(N11)

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 305  Solved: 105
[Submit][Status][Web Board]

Description

火车从始发站（称为第1站）开出，在始发站上车的人数为a,然后到达第2站，在第2站有人上、下车，但上、下车的人数相同，因此在第2站开出时（即在到达第3站之前）车上的人数保持为a人，从第3站起（包含第3站）上、下车的人数有一定的规律：上车的人数是前两站上车人数之和，而下车人数等于上一站上车人数，一直到终点站的前一站（n-1站），都满足此规律。现给出的条件是：共有N个车站，始发站上车的人数为a，最后一站下车的人数是m（全部下车）。访问从x站开出时车上的人数是多少？若无解输出“No answer.”（所有数据均<=2,000,000,000）

Input

第一个数为k(0＜k＜=20)，表示后面跟有k组测试数据，每组测试数据一行。每组测试数包括四个整数 a,n(n＞2),m和x(1＜=x＜n)。

Output

针对每组测试数据输出一个整数，表示x站开出时车上的人数。

Sample Input

2 1 6 7 3 1 6 7 4

Sample Output

2 4
