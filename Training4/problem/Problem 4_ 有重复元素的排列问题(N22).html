Problem 4: 有重复元素的排列问题(N22)

Description

设r={r1,r2,...,rn}是要进行排列的n个小写字母字符，其中字符r1,r2,...rn可能相同。给定n以及待排列的n个元素，设计一个算法，求出这n个字符的所有不同排列。

Input

第一行是元素个数n， 1＜=n＜=10，第二行是待排列的n个元素。

Output

n个元素的所有不同排列，最后一行中的数是排列总数

Sample Input

4 aacc

Sample Output

aacc acac acca caac caca ccaa 6
