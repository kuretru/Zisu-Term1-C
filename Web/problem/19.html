Problem 19: 计算阶乘

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 2  Solved: 2
[Submit][Status][Web Board]

Description

计算n的阶乘，n的阶乘为1*2*3*……*(n-1)*n。

Input

第一个数m（1< m < 10），表示有m组数据，及每组数据ni (0<= ni <15)输出。

Output

ni的阶乘，每组数据占一行。

Sample Input

3 0 2 6

Sample Output

1 2 720
