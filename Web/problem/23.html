Problem 23: 称质量

Problem 23: 称质量

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 33  Solved: 16
[Submit][Status][Web Board]

Description

用质量为1g、3g、9g、27g和81g的砝码(每个质量的砝码各一颗)称物体的质量，设砝码允许放在天平的两边，物体放在天平的左边。

Input

第一个数为n（n>0），表示有n个物体需称质量。接下来整数（以空格分格）表示每个物体的质量。

Output

称每个物体时砝码的放置，每个物体占一行。数字、left:、right:间分别以一个空格分隔。若不能称，则输出no answer

Sample Input

3 14 500 4

Sample Output

left: 1 3 9 right: 27 no answer left: right: 1 3
