Problem 12: 迭代方法求根

Description

利用一下所示的简单迭代方法求方程：cos(x)-x=0的一个实根。xn+1=cos(xn)迭代步骤如下：
（1）取x1的初值为0.0；
（2）把x1的值赋给x0（ 使用x0=x1 ）；
（3）求出一个新的x1（ 使用x1=cos(x0) ）；
（4）若x0-x1的绝对值小于0.000001执行步骤（计算绝对值可使用函数double fabs( double x )，头文件为<math.h>）；否则执行步骤（2）；
（5）程序将输出结果为：Root=0.739058。

Input

不需要输入

Output

Root=0.739085

Sample Input

Sample Output

Root=0.739085
