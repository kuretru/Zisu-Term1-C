Problem 26: 计算商和余数

Problem 26: 计算商和余数

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 18  Solved: 12
[Submit][Status][Web Board]

Description

输入两个字符串（长度不超过200），每个字符串占一行，并且都包含有数字符和其他字符。要求将字符串中的数字字符组成两个整数m和n(m,n<=2,000,000,000)，并输出这两个数商和余数。

Input

两个字符串，每个字符串占一行。

Output

商和余数，商和余数间使用一个空格分隔。

Sample Input

eyey4dyu5uid s6d@#$udisASDFD hd2.sd7h#$ gew

Sample Output

16 24
