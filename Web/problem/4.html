Problem 4: Factorial Problem

Problem 4: Factorial Problem

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 11  Solved: 8
[Submit][Status][Web Board]

Description

The factorial (阶乘) of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n. n!= 1*2*3*……*(n-1)*n. Please write a program to calculate the factorial of n.

Input

The input will consist of several numbers, separated by a space. The first number represents the total number m(1<m<10) of the non-negative integers to process. The following numbers represents all the m non-negative integers ni (0<= ni <15).

Output

There are m results. Display one result per line.

Sample Input

3 0 2 6

Sample Output

1 2 720
