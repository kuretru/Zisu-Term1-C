Problem 7: The Gas Station Problem

Problem 7: The Gas Station Problem

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 1  Solved: 1
[Submit][Status][Web Board]

Description

Sam is a student of ZISU. He usually drives home and could have access to n stations. The gas prices may vary across different gas stations. What's more, Every time Sam goes to the station to refill the car's gas tank, he will manage to top off the tank. Please write a program to calculate how much money Sam should take in order to arrive home successfully.

Input

The first line has a number k(0<k<20), where k means there are k groups of test data. In the following lines, input the test data that are separated by one blank line. In each group, the test data is organized as follows:
1. The first line includes one positive integer: n(1<=n<20), representing there are n gas stations that Sam could have access to.
2. The second line includes three parameters of the car's gas tank: gas tank capacity(liter), the amount of gas left in the tank and gas consumption per kilometer.
3. The following n lines have the data of all the n gas stations. Each line has two numbers: the distance(kilometer) between this station and the previous one Sam stopped by, and the gas price(dollars per liter).

4. The last line has one number representing the distance between the last gas station and Sam's house.

Output

There are k results for all test data. One result per line. The result is accurate to two decimal places(= shows two figures after the decimal point). If Sam cannot arrive home anyway, please display NO.

Sample Input

2 2 10 3 1 2 2 5 1 4 2 10 8 1 2 2 11 1 2

Sample Output

18.00 NO
