Problem 18: 平面分割

Problem 18: 平面分割

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 2  Solved: 2
[Submit][Status][Web Board]

Description

同一平面内有n(n<=500)条直线，已知其中有p(p>=2)条直线相交于一点，则这n条直线最多能将平面分割成多少个不同的区域？

Input

第一行输入一个整数m(m<=100)，表示有m组测试数据。接下来的m行，每行有两个整数n和p，分别表示平面内的直线数和相交于一点直线数。

Output

每组测试数据输出一个正整数，表示最多分割成的区域数目。每个测试数据的输出占一行。

Sample Input

2 3 2 12 5

Sample Output

7 73
