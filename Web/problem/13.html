Problem 13: 字符串处理

Problem 13: 字符串处理

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 6  Solved: 1
[Submit][Status][Web Board]

Description

给定一个字符串，将字符串下标为奇数的字符右移到下一个奇数位置，最右边将移出字符串的字符绕回放到第一个奇数位置，下标为偶数的字符不动（注：字符串的长度大于等于2）。

Input

从键盘读入1组字符串（长度小于100）

Output

交换位置后的1组字符串

Sample Input

abcdefghijk

Sample Output

kbadcfehgji
