Description

In number theory, a perfect number[完数]is a positive integer that is equal to the sum of all its positive, proper divisors[正的真因子，真因子是指比一个数小的因子]; for example, 6, which equals 1 + 2 + 3, and 28, which equals 1 + 2 + 4 + 7 + 14, are perfect numbers. Write a C program to output all the perfect numbers between 2 and 1000.

Input

none

Output

 Output all the perfect numbers between 2 and 1000.The output contains exactly t lines, each number to a line.
