Problem 2: Sin(x)

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 71  Solved: 34
[Submit][Status][Web Board]

Description

Given a number x, write a C program to calculate the value of sin(x). The precision[精度] of the result should be 1e-5. Note that the equation of cos(x) is as follows,
sin(x)=x-x^3/3!+x^5/5!-x^7/7!+...+x^(2n+1)/(2n+1)!+...
where x^(2n+1) is the (2n+1)(th) power[2n次方] of x and (2n)！is the factorial[阶乘] of (2n+1).

Input

Input a number x.

Output

Output  the value of  sin(x)  and correct to two decimal places.

Sample Input

0

Sample Output

0.00
