Problem 28: 猴子与香蕉(N30)

Problem 28: 猴子与香蕉(N30)

Time Limit: 1 Sec  Memory Limit: 128 MB
Submit: 26  Solved: 9
[Submit][Status][Web Board]

Description

科学家为了测试猴子的智商，将猴子关在一屋子里，屋子的天花板上挂着一只香蕉，但猴子够不到。屋子里面还有不规格的长方体若干，猴子可以将这些长方体叠起来，然后爬到长方体上去拿香蕉。但长方体的叠加是有要求的，叠在上面的长方体与长与宽必须大于叠在下面长方体的长和宽。现在要求你在给定的尺寸的长方体，帮助猴子计算出长方体叠加后的最大离地高度。假定每种尺寸的长方体都有3个。

Input

第一个数为m(0＜m＜=20)，表示后面跟有m组测试数据。每组测试数据的第一行为n(1＜=n＜=100)，表示有n种长方体，后面跟有n行数据，每行数据包括3个整数a1,a2,a3，分别表示长方体的长宽高。

Output

对于每组测试数据，计算并输出长方体叠加后的最大离地高度，每个输出占一行。

Sample Input

1 2 1 2 3 2 4 9

Sample Output

14
