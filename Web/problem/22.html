Problem 22: 数列

Problem 22: 数列

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 3  Solved: 3
[Submit][Status][Web Board]

Description

数列定义如下：
 f(1)=1, f(2)=1, f(n)=(A*f(n-1)+B*f(n-2)) mod 7
给定A，B和n，请计算f(n)

Input

包括多组测试数据，每组测试数据占一行，每组测试数据包括3个整数A，B和n（1<=A,B<=1000,1<=n<=100,000,000）。
  0 0 0表示测试数据结束。

Output

针对每组测试数据，输出f(n)的值，每个值占一行

Sample Input

1 1 3 1 2 10 0 0 0

Sample Output

2 5
