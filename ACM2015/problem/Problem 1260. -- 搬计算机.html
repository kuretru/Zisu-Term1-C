Problem 1260. -- 搬计算机

1260: 搬计算机

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 13  Solved: 4
[Submit][Status][Web Board]

Description

浙外ACM小组是成为出色程序员的摇篮，小明的梦想是成为一个伟大的计算机程序员。为此，他要求参加浙外ACM小组，老师为判断他的资质，给他出了一个难题。老师把他带到一个放满了计算机的实验室对他说：“小明，这个实验室里有一批不同的计算机，但你搬走每一台计算机都需要一些时间，每一台计算机也有他自身的计算速度。我会给你一段时间，在这段时间中，你可以搬走一些计算机。如果你是一个聪明的人，你应该可以让你搬走的计算机的计算速度之和达到最大。” 如果你是小明，你能完成这个任务吗？

Input

第一行有两个用空格分隔的整数T(1<=T<=1000)和M(1<=M<=100),T表示总共能够用来搬计算机的时间，M代表实验室中计算机的数量。接下来的M行每行包括两个1到100之间（包括1和100）的整数，分别表示搬这台计算机所需的时间和这台计算机的运行速度。

Output

一个整数，表示在规定时间内所搬走的计算机所达到运行速度之和的最大值。

Sample Input

70 3 71 100 69 1 1 2

Sample Output

3
