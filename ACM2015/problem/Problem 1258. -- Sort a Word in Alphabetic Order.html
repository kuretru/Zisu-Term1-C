Problem 1258. -- Sort a Word in Alphabetic Order

1258: Sort a Word in Alphabetic Order

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 15  Solved: 7
[Submit][Status][Web Board]

Description

Write a program to sort a word in alphabetic order: For example if the user enter a word "programming" then output will be "aggimmnoprr", that is, the output word will contain characters in alphabetical order. In our program we assume the input word contains only lower case alphabets.

Input

The input is a sequence of words containing only lower case alphabets and ending with the character #. Each two words are separated with a single space.

Output

The output should consist of the words sorted into alphabetical order.

Sample Input

online coding courses#

Sample Output

eilnno cdgino ceorssu
